Actores: Administrador

Disparador: Necesidad de acceso a zonas y atributos restringidos del sistema.

Descripción: Caso de Uso en donde se accede a la gestión de combo (Crear,Modificar,Eliminar).

Pre-condición: El administrador debe acceder a caracterìsticas restingidas del sistema.

Post-condición: El administrador debe seleccionar qué clase de gestión de combo desea realizar (Crear,Modificar,Eliminar).

El sistema brinda acceso al administrador. El sistema muestra en pantalla las distintas opciones de gestión de combo: Crear-Modificar-Eliminar. El administrador selecciona la opción deseada según su requirimiento.
