  • Actores: Administrador, Cliente y Usuarios externos.

  • Disparador: Necesidad de conocer productos y combos ofrecidos por la empresa.

  • Descripción: Caso de Uso en donde el usuario consulta el catálogo.

  • Pre-condición: El usuario debe estar ubicado en la página principal. El Administrador debe haber cargado los productos y combos disponibles.

  • Post-condición: Ninguna.

Flujo Normal

  1. El usuario ingresa al área de catálogos y ofertas.
  2. El sistema muestra los productos disponibles de la empresa.
  3. El usuario navega por los distintos productos ofrecidos con posibilidad de iniciar gestión de compra.
